# HUID

**Idea by [tsoding](https://www.youtube.com/watch?v=QH6KOEVnSZA&t=295s).**

A Human Unique Identifier (HUID) is a unique identifier with 1-second precision, optimized for human-scale interactions rather than high-frequency automated systems.
